# บทที่ 1: ดีบักเกอร์ผู้ไร้ค่า (The Null Debugger)

เสียงหายใจที่กระแทกขึ้นจากลำคอดังแข่งกับเสียงโลหะเสียดสีในโรงจอดรถใต้ดินร้าง คุริโตะถอยหลังหนึ่งก้าว ปลายรองเท้าสึกๆ ลื่นบนคราบน้ำมันและผงปูน จังหวะเดียวกันนั้นเงาดำสี่ขาก็พุ่งเข้ามาเหมือนภาพกระตุกในจอเสีย

มันไม่ใช่สุนัข และไม่ใช่สิ่งมีชีวิตที่โลกควรยอมรับว่ามีตัวตน

ร่างของมันเหมือนโครงกระดูกสุนัขที่ถูกหุ้มด้วยยางมะตอยเหลว สีดำเงาสะท้อนแสงไฟฉุกเฉินเป็นคลื่นบิดเบี้ยว ตลอดแนวลำตัวมีตัวเลข 0 กับ 1 สีแดงไหลวนอยู่ใต้ผิวเหมือนหนอนเรืองแสง มันอ้าปากกว้างเกินกายภาพจริง เขี้ยวสีขาวพร่าแตกออกเป็นพิกเซลก่อนจะประกอบกลับเข้าที่อีกครั้ง

คุริโตะยกท่อเหล็กขึ้นรับตามสัญชาตญาณ

เคร้ง!

แรงปะทะกระแทกเข้าข้อมือจนชาทั้งแขน เขากัดฟันแน่น แต่บั๊กหมาป่ากลิซตรงหน้ากลับเพียงชะงักไปครึ่งวินาที ดวงตาหกดวงของมันขยับเป็นจังหวะไม่พร้อมกันอย่างน่าขยะแขยง ก่อนกรงเล็บจะปาดสวนลงมารวดเร็วราวมีคนกดเร่งภาพ

คุริโตะทิ้งตัวเฉียงหลบ เล็บคมเฉี่ยวไหล่ซ้ายจนเสื้อขาด เลือดร้อนๆ ไหลซึมลงตามแขนทันที

"ล่อมันดีๆ สิวะ ไอ้เลเวลศูนย์!" เสียงตะโกนดังมาจากด้านหลัง "แค่นี้ยังทำไม่ได้ แล้วจะเอาเงินไปใช้หนี้ได้ยังไง"

คนพูดคือทานากะ ดีบักเกอร์ Level 1 จากเซิร์ฟเวอร์ Iron Wall สาขาย่อยฝั่งตะวันตก ชายร่างใหญ่ในเกราะคอมโพสิตสีเงินที่พ่นตราโลโก้บริษัทรับเหมาเอาไว้เต็มอก เขาถือดาบโปรแกรมใบยาวที่ปล่อยแสงสีฟ้าจากรอยสลักตลอดแนวคมดาบ อาวุธหนึ่งชิ้นในมือผู้ชายคนนั้นมีราคาแพงกว่าค่าเช่าห้องของคุริโตะทั้งปี

คุริโตะไม่ได้หันไปมอง เขารู้ดีว่าตัวเองเป็นแค่แรงงานชั่วคราวที่ถูกจ้างมาให้เสี่ยงตายแทนคนถืออาวุธจริง

ในโลกหลังเหตุการณ์ Blue Screen Event คนอย่างเขาถูกเรียกหลายชื่อ ทั้งของเสีย ทั้ง Null ทั้ง Error เดินได้

ค่าการซิงค์ของเขาอยู่ที่ 0.51 เปอร์เซ็นต์ ต่ำจนเดอะ เคอร์เนลแทบไม่ยอมรับว่าเขาเชื่อมต่อกับโลกชั้นในได้ด้วยซ้ำ

หน้าที่ของเขาในภารกิจแบบนี้จึงไม่ใช่การกำจัดบั๊ก แต่เป็นการทำตัวให้มีเสียงดังพอจะดึงความสนใจของมันไว้ชั่วคราว

คุริโตะลากท่อเหล็กครูดพื้นคอนกรีตให้เกิดเสียง ก้องสะท้อนทั่วชั้นจอดรถที่แตกร้าว

"มานี่สิ ไอ้ตัวเพี้ยน"

กลิซหมาป่าหันมาทันที มันโก่งหลังต่ำแล้วพุ่งอีกครั้ง

คนทั่วไปอาจเห็นเพียงเงาดำวาบหนึ่ง แต่สายตาของคุริโตะเห็นมากกว่านั้น เขาเห็นเส้นทางการเคลื่อนที่ของมันเป็นริ้วข้อมูลสีแดง เห็นเศษโค้ดแตกพร่าออกจากปลายเล็บก่อนถึงตัวจริงครึ่งจังหวะ ความสามารถประหลาดนี้ไม่เคยทำให้เขาเก่งขึ้น มันแค่ทำให้เขารอดตายมาได้นานกว่าที่คนอื่นคาด

ขวาล่าง

สัญชาตญาณของเขากระตุก คุริโตะย่อตัวลงโดยไม่คิด ร่างหมาป่ากลิซพุ่งเฉียดหัวไปเพียงเส้นผม ลมจากการเคลื่อนที่ฟาดใบหน้าเขาจนแสบ

"ตอนนี้!" ทานากะตะโกน

ดาบโปรแกรมแทงทะลุสีข้างของบั๊กจากด้านหลัง แสงฟ้าพุ่งออกจากบาดแผลเป็นเส้นเฉียบคม ร่างสัตว์ประหลาดเกร็งค้าง ส่งเสียงร้องคล้ายคลื่นรบกวนในลำโพงเก่า แล้วทั้งตัวก็แตกออกเป็นสี่เหลี่ยมแสงที่ละลายหายไปในอากาศ

ผลึกซอร์สแฟรกเมนต์สีฟ้าขุ่นหล่นกระทบพื้นดังแกร๊ก

เสียงเย็นชาไร้อารมณ์ดังขึ้นในหัวทุกคน

**[Kernel Notice: Bug removed. Source fragment registered.]**

ทานากะเดินเข้ามาเก็บผลึกใส่กระเป๋าอย่างรวดเร็ว ราวกลัวว่าคุริโตะจะกล้าเถียงสิทธิ์ของตัวเอง ทั้งที่รู้อยู่เต็มอกว่าแรงงานชั่วคราวไม่มีสิทธิ์แม้แต่แตะของดรอป

"ช้ากว่านี้อีกนิด ข้าคงต้องเปลืองแพตช์ฮีลเพิ่ม" เขาพูดพลางใช้ปลายรองเท้าแตะต้นขาคุริโตะเบาๆ อย่างดูถูก "นี่ ส่วนของแก"

ธนบัตรเครดิตเก่าๆ ถูกโยนลงบนพื้นใกล้มือ

สามพันเครดิต

ค่าเสี่ยงตายในครึ่งวัน เท่ากับยาฉีดแอนติไวรัล เซรั่มของคารินไม่ถึงหนึ่งในสิบโดส

คุริโตะก้มลงเก็บเงินช้าๆ มือเขาสั่นจากความเหนื่อยและเสียเลือด แต่เขายังพยายามยิ้มให้เหมือนคนที่ไม่เดือดร้อนมากนัก

"ขอบคุณครับ"

ทานากะหัวเราะหยัน "ถ้าหน้าตาแกดูมีค่ากว่านี้ เซิร์ฟเวอร์คงไม่ให้มาล่อบั๊กแล้วหรอก ไปทำแผลเองด้วย อย่ามาเบิกของ Iron Wall"

คุริโตะไม่ตอบ เขาแค่ยืนขึ้น ช้อนสายเป้ที่ขาดเกือบหลุดขึ้นพาดบ่า แล้วเดินออกจากเดดเซกเตอร์ย่อยนั้นพร้อมกลิ่นเลือดติดตัว

ประตูนิรภัยพาเขากลับออกสู่โลกจริงพร้อมแรงกดเบาๆ ในหู ราวกับผ่านผิวน้ำขึ้นมาหายใจอีกครั้ง

ด้านนอกเป็นยามเย็นของนีโอ-โตเกียว แสงโฆษณาโฮโลกราฟิกจากตึกสูงสะท้อนผ่านฝนละอองบางๆ บนท้องถนน ด้านบนสุดของเส้นขอบฟ้า โดมไฟร์วอลล์เหนือเมืองกะพริบวูบเป็นจังหวะผิดปกติ สีฟ้าของมันซีดเหมือนจอที่โดนไฟกระชากมาหลายรอบ

คุริโตะก้มหน้าเดินผ่านผู้คนที่แต่งตัวสะอาด สวมตราเซิร์ฟเวอร์ดังบนอกเสื้อ หรือไม่ก็เป็นพนักงานบริษัทข้อมูลระดับกลางที่ยังพอเชื่อว่าระบบของโลกนี้ควบคุมได้อยู่

จอข่าวขนาดยักษ์บนอาคารขนส่งฉายภาพรายงานสดจากอีกฟากโลก

เซาเปาโลพบเดดเซกเตอร์ใต้แม่น้ำเพิ่มอีกสองจุด

มุมไบประกาศปิดสนามบินชั่วคราวหลังรันเวย์เกิดการกลายเป็นพิกเซล

โซลกำลังถกเถียงเรื่องสิทธิแรงงานของดีบักเกอร์ Level 0 ที่ถูกใช้เป็นเหยื่อล่อโดยไม่คุ้มครองตามกฎหมาย

คุริโตะเงยหน้ามองแค่ครู่เดียวแล้วเดินต่อ ข่าวพวกนั้นยิ่งใหญ่เกินชีวิตเขา โลกจะพังทั้งใบหรือไม่ ตอนนี้สิ่งที่เขาต้องทำมีแค่หาเงินไปต่ออายุชีวิตน้องสาวให้ครบอีกสัปดาห์

เขาแวะร้านสะดวกซื้อยี่สิบสี่ชั่วโมง ซื้อข้าวปั้นลดราคาใกล้หมดอายุ บะหมี่กึ่งสำเร็จรูปสองซอง และพลาสเตอร์ปิดแผลแบบถูกที่สุด พนักงานแคชเชียร์เหลือบเห็นเลือดซึมที่ไหล่เขา แต่ก็ไม่ถามอะไร เมืองนี้ชินกับคนเจ็บที่ไม่มีเงินไปหาหมอแล้ว

จากร้านสะดวกซื้อ คุริโตะไม่กลับห้อง เขาเปลี่ยนเส้นทางไปโรงพยาบาลกลางเขตฮาจิโอจิทันที

กลิ่นน้ำยาฆ่าเชื้อกับเสียงระบบระบายอากาศต้อนรับเขาเหมือนทุกคืน แสงสีขาวของโถงทางเดินสะอาดเกินจะเข้ากับรองเท้าผ้าใบเปื้อนเลือดของเขา

พยาบาลหน้าเคาน์เตอร์การเงินชื่อซาโตะจำเขาได้ทันที เธอชะงักนิดหนึ่งเมื่อเห็นแผลใหม่บนไหล่

"คุณคุริโตะ ไปทำแผลก่อนดีไหมคะ"

"ไม่เป็นไรครับ ขอชำระก่อน"

เขาวางธนบัตรทั้งหมดลงบนเคาน์เตอร์ เก็บไว้กับตัวเพียงไม่กี่ร้อยสำหรับค่ารถและอาหารวันพรุ่งนี้

ซาโตะรับเงินไปนับ หน้าจอใสตรงหน้าเธอเลื่อนตัวเลขค้างชำระขึ้นมาเป็นทางยาว เธอเม้มปากเล็กน้อยก่อนพูดด้วยน้ำเสียงที่อ่อนลงอย่างชัดเจน

"หลังหักยอดนี้แล้ว ค่ารักษาคุณคารินยังคงค้างอีกหนึ่งแสนสี่หมื่นเจ็ดพันเครดิตค่ะ"

คุริโตะพยักหน้า เขารู้ตัวเลขนั้นดีอยู่แล้ว แต่อ่านออกจากปากคนอื่นทีไร มันเหมือนมีของมีคมค่อยๆ กรีดใต้ซี่โครงทุกครั้ง

"แล้วค่ายาเข็มถัดไป..."

"ภายในห้าวันค่ะ" ซาโตะหยุดเล็กน้อยก่อนพูดต่อ "แอนติไวรัล เซรั่มล็อตใหม่แพงขึ้น เพราะโรงงานผลิตที่โอซาก้าโดนปิดจากเหตุข้อมูลปนเปื้อน"

คุริโตะหลับตาอยู่ครู่หนึ่ง

แม้แต่ยาที่ใช้ยื้อชีวิตคารินก็ยังแพงขึ้นตามจังหวะที่โลกพังลงทีละส่วน

"ผมจะหาให้ทัน"

เขาพูดประโยคนั้นเหมือนเดิมทุกสัปดาห์ จนไม่แน่ใจแล้วว่ากำลังให้คำสัญญากับคนอื่น หรือกำลังหลอกตัวเองอยู่

ประตูห้อง 302 เปิดออกอย่างเงียบเชียบ

คารินนอนอยู่บนเตียงโดยมีแสงจากเครื่องมอนิเตอร์ฉาบผิวซีดของเธอให้ดูโปร่งแสงยิ่งกว่าเดิม เด็กสาววัยสิบหกที่ควรได้ไปโรงเรียน กลับต้องนอนอยู่ในห้องปลอดเชื้อและดูเวลาชีวิตหายไปทีละพิกเซล

นิ้วก้อยกับนิ้วนางมือซ้ายของเธอหายไปแล้ว เหลือเพียงเศษแสงสี่เหลี่ยมจางๆ ลอยค้างเป็นช่วงๆ เหมือนระบบยังลังเลว่าจะลบส่วนนี้ของร่างออกจนหมดเมื่อไร ผิวบริเวณข้อมือมีเส้นแตกละเอียดเหมือนกระจกหน้าจอที่ถูกกดแรงเกินไป

คุริโตะวางถุงร้านสะดวกซื้อไว้บนโต๊ะ แล้วเดินไปนั่งข้างเตียง เขาแตะหลังมือขวาของน้องสาวเบาๆ มือเธอเย็นกว่าทุกครั้ง

คารินลืมตาขึ้นช้าๆ เมื่อสัมผัสได้ถึงเขา

"พี่มาแล้วเหรอ"

เสียงเธอยังนุ่มเหมือนเดิม แต่แผ่วจนแทบเป็นลมหายใจ

"อือ วันนี้รถติดนิดหน่อย" เขาโกหกโดยอัตโนมัติ ไม่อยากเล่าเรื่องเลือด เรื่องบั๊ก เรื่องที่ตัวเองเกือบโดนผ่าครึ่งให้เธอฟัง

คารินเหลือบเห็นแผลที่ไหล่เขาอยู่ดี เด็กสาวขมวดคิ้วทันที

"พี่ไปทำงานในเซกเตอร์อีกแล้วใช่ไหม"

"งานเบาๆ เอง"

"พี่โกหกไม่เก่งเลย"

เธอพยายามยิ้ม แต่รอยยิ้มนั้นทำให้คุริโตะอยากหันหน้าหนีมากกว่าเดิม เขาเปิดข้าวปั้น หยิบชิ้นปลาแซลมอนเล็กๆ ที่พอมีเนื้อจริงอยู่บ้างวางใส่จานให้คารินก่อน แต่อีกฝ่ายส่ายหน้า

"พี่กินเถอะ ฉันเพิ่งกินของโรงพยาบาลไป"

"ของโรงพยาบาลนั่นเรียกว่าอาหารได้ด้วยเหรอ"

คารินหัวเราะแผ่วๆ จากนั้นมองออกไปนอกหน้าต่าง กระจกสะท้อนแสงโดมไฟร์วอลล์สีฟ้าซีดอยู่ไกลๆ

"วันนี้ฉันฝันถึงแม่อีกแล้ว" เธอพูด "แม่ยืนอยู่ตรงสถานีรถไฟเดิมตอนเกิด Blue Screen Event แล้วบอกว่าอย่าหลับนานเกินไป"

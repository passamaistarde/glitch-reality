# บทที่ 2: กับดักในเขต D-13 (The Trap in D-13)

ความมืดภายใน D-13 หนักแน่นจนเหมือนมีน้ำหนักกดทับอยู่บนเปลือกตา

คบเพลิงเคมีในมือคุริโตะปล่อยแสงสีเขียวซีดออกมาเป็นวงแคบๆ ไม่พอจะไล่ความดำของอุโมงค์รถไฟเก่าได้หมด ผนังคอนกรีตทั้งสองฝั่งเปียกชื้นและปริแตก มีเส้นเคเบิลเก่าห้อยระโยงเหมือนไส้ของซากเครื่องจักรยักษ์ รางเหล็กด้านล่างขึ้นสนิมแดงจัดจนดูคล้ายเส้นเลือดที่แข็งตายแล้ว

ปกติเดดเซกเตอร์ระดับต่ำแบบนี้ควรมีเสียงจิ๊ดๆ ของ Spark Rat วิ่งไปมา ควรมีแสงไฟช็อตเป็นพักๆ หรืออย่างน้อยก็ต้องมีคราบเถ้าจากบั๊กที่เพิ่งถูกกำจัด

แต่ที่นี่เงียบ

เงียบเกินไป

เสียงเดียวที่ชัดคือฝีเท้าของพวกเขา และเสียงลมหายใจของโมริที่เริ่มสั้นถี่ขึ้นทุกที

"คุณหนุ่ม" ชายชรากระซิบเรียกจากด้านหลัง "ผมไม่ชอบที่นี่เลย"

คุริโตะไม่หันไปมอง เขากำลังเพ่งสายตาผ่านอากาศมืดข้างหน้า ตัวเลข 0 กับ 1 สีแดงยังลอยอยู่เหมือนฝุ่น แต่ตอนนี้มันชัดเจนกว่าเดิม มันไหลเป็นริ้วจากผนัง จากเพดาน จากรางเหล็ก รวมตัวกันเป็นกระแสเดียวราวแม่น้ำข้อมูลที่กำลังมุ่งสู่ใจกลางบางอย่างในส่วนลึก

เขาไม่เคยเห็นปรากฏการณ์แบบนี้มาก่อน

ตามปกติ โค้ดของโลกที่เขามองเห็นจะซ้อนทับกันอย่างเลือนรางเหมือนแสงสะท้อนในกระจก แต่ใน D-13 เส้นข้อมูลกลับแน่นขึ้น กระเพื่อมเร็วขึ้น และที่น่ากลัวที่สุดคือมันมีเจตนา

"หยุดคุยกันแล้วเดิน" ไทกะพูดโดยไม่หันกลับมา "ใกล้ถึงโถงหลักแล้ว"

เคนจิเดินประกบหัวหน้าทีม มือขวากำปืนเวกเตอร์แน่น ส่วนมิกะเคลื่อนไหวเงียบเชียบอยู่ริมผนังเหมือนเธอคุ้นกับความมืดมากกว่าคนทั่วไป

คุริโตะสังเกตว่าทั้งสามคนไม่ใช่ทีมที่ไว้ใจกันจริง เคนจิหันมองไทกะบ่อยเกินจำเป็น มิกะเองก็เผลอแตะบางอย่างที่ซ่อนไว้ในกระเป๋าเสื้อเป็นระยะ คล้ายเช็กอุปกรณ์ลับมากกว่าจะเตรียมรับศัตรู

"ค่าจ้างสูงแบบนี้ คงไม่ได้มาขุดแค่แร่ข้อมูลหรอก" คุริโตะลองหยั่งเชิง

ไทกะหัวเราะในลำคอ "รู้มากแล้วได้อะไร ถ้าออกไปถึงข้างนอก เดี๋ยวก็ได้เงินเท่าที่ตกลงเอง"

คำตอบนั้นยิ่งทำให้อากาศเย็นลงอีกชั้น

อุโมงค์เปิดเข้าสู่ชานชาลากว้างของสถานีเก่า ป้ายชื่อสถานีด้านบนพังเหลือเพียงตัวอักษรบางส่วน ไฟโฆษณาจากยุคก่อน Blue Screen Event ยังคงค้างอยู่ในสภาพเสียครึ่งหนึ่ง มีภาพหญิงสาวถือโทรศัพท์รุ่นโบราณยิ้มค้าง ใบหน้าถูกบั๊กกินไปครึ่งแก้มจนเป็นช่องสี่เหลี่ยมดำ

พื้นชานชาลาเกลื่อนไปด้วยซาก Spark Rat หลายสิบตัว

พวกมันไม่สลายเป็นฝุ่นข้อมูลตามธรรมชาติ แต่แห้งฝ่อเหมือนถูกดูดพลังงานออกจนเหลือเพียงเปลือก เส้นแสงด้านในกลวงโบ๋ บางตัวมีรอยเจาะสองรูที่ลำคอ บางตัวถูกฉีกครึ่งแต่กลับไม่มีซอร์สแฟรกเมนต์เหลือแม้แต่นิดเดียว

โมริกลืนน้ำลาย "นี่มันไม่ใช่ระดับต่ำแล้ว"

คุริโตะย่อตัวลงมองซากหนูตัวหนึ่งใกล้เท้า ตัวเลขสีแดงบนร่างมันยังพริบเป็นช่วงๆ แล้วดับเหมือนแบตเตอรี่หมด ข้างซากมีคราบบางอย่างสีรุ้งคล้ายเลือดทีวีแตกซ่านแห้งติดพื้นอยู่เป็นแถบ

"Vampire Bat Glitch" มิกะพึมพำ เธอใช้ลวดที่ข้อมือเขี่ยซากขึ้นมาดู "รอยดูดซอร์สแบบนี้ใช่แน่"

เคนจิสบถทันที "มันมาอยู่ใน D-13 ได้ยังไง"

ไทกะไม่ตอบ เขาหรี่ตามองลึกเข้าไปในชานชาลาร้าง แล้วหยิบอุปกรณ์ทรงกลมขนาดเท่าลูกเบสบอลออกมาจากกระเป๋าเสื้อเกราะ มันสลักสัญลักษณ์บริษัทเวชชีวะที่คุริโตะไม่คุ้นชื่อ

คุริโตะขมวดคิ้ว "นั่นอะไร"

ไทกะใส่มันกลับเข้ากระเป๋าทันที "ไม่เกี่ยวกับนาย"

ความเงียบกดทับลงมาอีกครั้ง ก่อนที่ทุกอย่างจะระเบิดออกพร้อมกัน

พรึ่บ!

เงาดำนับร้อยเทตัวลงจากเพดานสถานีเหมือนม่านสดสีดำ ดวงตาสีแดงเล็กๆ นับไม่ถ้วนเปิดขึ้นในความมืด เสียงกรีดร้องแหลมสูงทิ่มทะลุเข้าในหูจนคุริโตะรู้สึกเหมือนแก้วหูฉีก

"ตั้งแนว!" ไทกะคำราม "เคนจิ ยิงสกัด! มิกะ ตัดด้านซ้าย!"

ปืนเวกเตอร์ของเคนจิปล่อยกระสุนแสงสีน้ำเงินออกไปเป็นชุดยาว กลางอากาศเกิดจุดระเบิดแตกพร่า ฝูงค้างคาวหลายตัวสลายเป็นพิกเซลทันที แต่มันมีมากเกินไป มากเสียจนเหมือนความมืดเองกำลังมีฟันและปีก

มิกะสะบัดลวดโลหะออกไปเป็นวง คมลวดตัดค้างคาวสามตัวขาดกลางลำตัว ทว่าเลือดสีรุ้งที่แตกออกมากลับกลายเป็นละอองที่ยิ่งทำให้มุมมองสับสนกว่าเดิม

โมริชะงักเพียงครึ่งจังหวะครึ่งเดียว และครึ่งจังหวะนั้นก็พอให้เขาตาย

ค้างคาวกลิซตัวหนึ่งเกาะลงบนใบหน้าชายชรา เขาร้องลั่นพร้อมปล่อยคบเพลิงหลุดมือ วินาทีต่อมาค้างคาวอีกหลายสิบตัวก็ถาโถมเข้าใส่เหมือนผ้าใบสีดำขยับได้ ร่างของโมริล้มลงกับพื้น ถูกรุมทึ้ง ลากหายเข้าใต้บันไดเลื่อนที่พังแล้ว เสียงขอความช่วยเหลือของเขายาวอยู่ไม่ถึงห้าวินาที

คุริโตะตัวเย็นเฉียบ

เขารู้ตัวว่าไม่ได้ชินกับความตายเลย แค่ถูกบังคับให้มองมันบ่อยจนทำหน้าเฉยได้เท่านั้น

ค้างคาวสองตัวพุ่งเข้าหาเขาจากด้านขวา เส้นข้อมูลสีแดงในสายตากรีดเป็นมุมเฉียบ เขาเหวี่ยงท่อเหล็กสวนตามแนวที่มองเห็น

เคร้ง!

ตัวแรกโดนฟาดกระเด็นใส่ผนัง อีกตัวเฉี่ยวผ่านไหล่ขวาไป ฉีกผ้าคลุมแขนขาดเป็นริ้ว ท่อเหล็กสั่นสะท้านอยู่ในมือเขาจนแทบหลุด แต่เขายังหมุนตัวอาศัยแรงส่งเตะคบเพลิงตกแตกใส่ฝูงตรงหน้า ทำให้เกิดแสงวาบและควันเคมีคละคลุ้งชั่วครู่

"ไอ้เลเวลศูนย์ ถอยมาทางนี้!" เคนจิตะโกน

คำเตือนนั้นมาช้าไปนิดหนึ่ง ค้างคาวตัวหนึ่งฉีกผ่านแนวปืนของเคนจิแล้วกัดเข้าที่ปลายแขนเขาเต็มๆ

"อ๊าก!"

รอยกัดเรืองแสงแดงสดก่อนกระจายเป็นลายเส้นเหมือนไฟล์เสีย แขนของเคนจิเริ่มแตกเป็นบล็อกพิกเซลจากข้อมือขึ้นไป เขาหน้าซีดเผือด รีบเอามืออีกข้างบีบเหนือแผลด้วยสีหน้าตื่นตระหนก

ไทกะหันไปมองเพียงแวบเดียวแล้วตัดสินใจทันที

"ถ้าแพร่ขึ้นข้อศอก ตัดแขนทิ้ง"

"บ้าไปแล้ว!" เคนจิตวาดกลับ "แกเป็นคนบอกว่างานนี้ปลอดภัย!"

เสียงนั้นหลุดออกมาแบบคนที่ความกลัวฉีกหน้ากากความจงรักภักดีทิ้งเรียบร้อยแล้ว

คุริโตะจับประโยคสำคัญได้พอดี

"แกเป็นคนบอก"

งั้นอย่างน้อยไทกะก็รู้อะไรมากกว่าที่พูด

ไทกะสบถ ขณะฟันดาบโปรแกรมกวาดศัตรูรอบตัวออกไป "ฉันแค่รู้ว่ามีรังพิเศษซ่อนอยู่ ไม่รู้ว่ามันจะเข้าสู่ช่วงฟักตัวเร็วขนาดนี้!"

มิกะหันขวับ "นายบอกแค่ว่าจะมีโบนัสถ้าเอาควีนคอร์กลับไปได้ ไม่ได้บอกว่าจะต้องใช้เหยื่อล่อสดจริงๆ"

คำว่าเหยื่อล่อสดทำให้เลือดในตัวคุริโตะเย็นกว่าความมืดรอบข้าง

เขานึกถึงข้อความแปลกเมื่อคืนทันที

fresh carriers required

lure phase

มันไม่ใช่ภาพหลอน

พวกเขากับโมริถูกจ้างมาไม่ใช่เพื่อแบกของ แต่เพื่อเป็นชิ้นส่วนในพิธีล่ออะไรบางอย่างให้ออกจากรัง

"ไอ้สารเลว" คุริโตะกัดฟัน

ไทกะไม่แม้แต่จะเสียเวลาหันมาสนใจ เขากดฝ่ามือลงกับพื้นชานชาลาแล้วเอ่ย Command

"[Patch Command: Blade Drift]"

อักขระสีน้ำเงินพุ่งออกจากฝ่ามือเขาเป็นวง ครู่เดียวใบมีดพลังงานหลายสิบเล่มก็ลอยขึ้นจากพื้นแล้วพุ่งกวาดรอบตัวเหมือนพายุคมมีด ฝูงค้างคาวจำนวนมากถูกผ่าแหลก ช่องว่างจึงเปิดขึ้นกลางชานชาลา

"วิ่ง!" ไทกะตะโกน "ไปอุโมงค์ฝั่งเหนือ!"

มิกะลังเลเพียงวินาที ก่อนจะวิ่งตามหัวหน้าทีม เคนจิกัดฟันกอดแขนที่กำลังถูกกลืนกินไว้แล้วพุ่งตามไปเช่นกัน

คุริโตะไม่ขยับ

ตัวเลขสีแดงทั้งหมดในอากาศกำลังไหลไปทางอุโมงค์ฝั่งเหนือเหมือนแม่น้ำบ้าที่หิวโหย กระแสของมันหนาแน่นจนปลายทางดูเหมือนกำแพงเลือดเรืองแสงที่คนทั่วไปมองไม่เห็น

"คุริโตะ!" มิกะหันมาตะโกน เธอเรียกชื่อเขาเป็นครั้งแรก "รีบมา!"

"ทางนั้นไม่ใช่ทางหนี" เขาตอบ เสียงตัวเองแหบจนแทบจำไม่ได้ "มันเป็นปากของกับดัก"

เคนจิหัวเราะอย่างคนจะบ้า "แกเพี้ยนไปแล้วหรือไง"

ไทกะเลือกไม่เสียเวลา เขาวิ่งนำหายเข้าไปในอุโมงค์มืดทันที พร้อมเสียงรองเท้าของอีกสองคนที่ตามหลังติดๆ

คุริโตะถอยหลังไปชิดตู้จำหน่ายตั๋วเก่าที่พังครึ่งหนึ่ง มือกำท่อเหล็กแน่นจนข้อนิ้วซีดขาว ฝูงค้างคาวที่เหลือวนอยู่เหนือศีรษะเขา แต่แปลกประหลาดตรงที่พวกมันเริ่มไม่สนใจเหยื่อใกล้มืออีกต่อไป พวกมันทั้งหมดหันไปทางอุโมงค์เดียวกับที่สามคนนั้นวิ่งหนี เหมือนกำลังรอฟังสัญญาณจากบางสิ่งที่สูงกว่าในห่วงโซ่เดียวกัน

แล้วพื้นก็สั่น

ไม่ใช่แรงสั่นจากอาคารเก่าทรุดตัว แต่เป็นแรงสะเทือนลึกจากข้างใต้ เหมือนมวลขนาดมหึมากำลังขยับตัวหลังหลับมานาน

ติ๊ง

เสียงแจ้งเตือนดังในหัวคุริโตะ คราวนี้ชัดเจนจนเขาแทบสำลักอากาศ

**[Kernel Warning: Hidden Layer protocol confirmed.]**

**[Entering unauthorized zone: The Nest.]**

**[User permission: insufficient.]**

เขาเบิกตาค้าง

ตลอดหลายปีที่ผ่านมา เดอะ เคอร์เนลมักส่งเสียงแตกพร่าเหมือนสัญญาณวิทยุเสีย เขาได้ยินเป็นคำไม่กี่คำแล้วหาย แต่ตอนนี้มันกลับชัดราวกับระบบทั้งโลกหันมาพูดกับเขาโดยตรง

เสียงกรีดร้องดังขึ้นจากอุโมงค์ฝั่งเหนือ

เริ่มจากเคนจิ ตามด้วยมิกะ และจบด้วยเสียงคำรามด่าทอของไทกะที่ขาดหายไปกลางประโยค ราวกับมีบางอย่างปิดทับปากพวกเขาทั้งเป็น

จากนั้นความเงียบก็ถล่มลงมาอีกครั้ง

คุริโตะรู้แล้วว่าตัวเองคิดถูกเกินไป และการคิดถูกในสถานการณ์แบบนี้ไม่ได้ช่วยให้รอดเลยแม้แต่น้อย

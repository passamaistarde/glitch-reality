# คัมภีร์โลกและการตั้งค่า (World Bible & Settings)

**Project Code:** Glitch Reality (ชื่อชั่วคราว)
**Theme:** Urban Fantasy / System Error / Cyber-Occult

## 1. แก่นเรื่อง (Core Concept)
โลกไม่ได้ถูกรุกรานโดยปีศาจจากต่างมิติ แต่ **"ความจริง" (Reality)** ของโลกกำลัง **"พังทลาย" (Glitch)** เหมือนโปรแกรมคอมพิวเตอร์ที่โค้ดเสียหาย สิ่งที่มนุษย์เรียกว่า "ดันเจี้ยน" คือไฟล์ขยะ (Corrupted Data) ที่แทรกซึมเข้ามาในเลเยอร์ของโลกความเป็นจริง

## 2. ศัพท์เฉพาะ (Terminology)

| คำศัพท์ทั่วไป (Common Term) | คำศัพท์ในเรื่อง (Unique Term in Novel) | ความหมาย / รายละเอียด |
| :--- | :--- | :--- |
| **Hunter (ฮันเตอร์)** | **Debugger (ดีบักเกอร์)** หรือ **Fixer (ฟิกเซอร์)** | ผู้ที่มีความสามารถในการมองเห็น "โค้ด" ของโลกและแก้ไขความผิดปกติ ไร้ซึ่งพลังเวท แต่ใช้การ "เขียนทับ" หรือ "ลบ" ข้อมูล |
| **Dungeon (ดันเจี้ยน)** | **Dead Sector (เดดเซกเตอร์)** | พื้นที่ที่ข้อมูลเสียหาย โลกภายนอกจะมองไม่เห็น คนธรรมดาที่หลงเข้าไปจะถูกลบตัวตน (De-rezzed) |
| **Monster (มอนสเตอร์)** | **Bug (บั๊ก)** หรือ **Glitch (กลิซ)** | สิ่งมีชีวิตที่เกิดจากข้อมูลขยะ มีรูปร่างบิดเบี้ยว ไม่สมประกอบ เลือดเป็นสีรุ้งแตกซ่าเหมือนจอทีวีเสีย |
| **Mana (มานา)** | **Source Fragments (ซอร์สแฟรกเมนต์)** | เศษข้อมูลที่ตกหล่นเมื่อกำจัด Bug ได้ ใช้สำหรับอัปเกรดหรือ "Patch" ร่างกาย |
| **System (ระบบ)** | **The Kernel (เดอะ เคอร์เนล)** | เสียงในหัวหรือหน้าต่างข้อความที่คอยแจ้งเตือน แต่มันมักจะทำงานผิดพลาด แจ้ง Error Code แทนที่จะบอกตรงๆ |
| **Guild (กิลด์)** | **Server (เซิร์ฟเวอร์)** | กลุ่มของ Debugger ที่รวมตัวกันคุมพื้นที่เขตข้อมูลต่างๆ |
| **Awakening (การตื่นรู้)** | **Synchronization (การซิงค์)** | กระบวนการที่มนุษย์เชื่อมต่อกับ The Kernel ได้สำเร็จ |

## 3. ระดับความสามารถ (Hierarchy Matrix)
ถูกวัดค่าด้วย **"Synchronization Rate" (ค่าซิงค์)** หรือ **"Kernel Level"**

1.  **Level 0 (Null User):** ผู้มีค่าซิงค์ต่ำเตี้ยเรี่ยดิน (0.01% - 0.99%) แทบจะไร้พลัง มักถูกดูถูกว่าเป็น "Error" หรือ "Glitch" ที่ไร้ค่า
2.  **Level 1 (User):** ดีบักเกอร์ระดับล่าง ทำได้แค่ต่อสู้กายภาพ ใช้เครื่องมือพื้นฐาน
3.  **Level 2 (Moderator):** ดีบักเกอร์ระดับกลาง เริ่มใช้คำสั่ง (Command) แก้ไขพื้นที่รอบตัวได้เล็กน้อย
4.  **Level 3 (Admin):** ดีบักเกอร์ระดับสูง เปลี่ยนกฎฟิสิกส์ใน Dead Sector ได้
5.  **Level 4 (Game Master / GM):** ระดับตำนานที่ควบคุม Sector ได้เหมือนบ้านตัวเอง
6.  **Anomaly (Unknown):** ตัวตนที่วัดค่าไม่ได้ ระบบไม่สามารถประมวลผล (พระเอกในอนาคต)

## 4. ตัวละครหลัก (Characters) & ชื่อ (Naming Convention)
ใช้ชื่อสไตล์ญี่ปุ่นสมัยใหม่ (Modern Edgy) ตัวละครหลากหลายเชื้อชาติ

### ตัวเอก (Protagonist)
*   **ชื่อ:** **คุริโตะ (Kurito)**
    *   *บทบาท:* ชายหนุ่มผู้แบกรับภาระค่ารักษาพยาบาลมหาศาลของน้องสาว เป็นดีบักเกอร์ "Level 0" ที่อ่อนแอที่สุดในประวัติศาสตร์
    *   *ลักษณะ:* ดูธรรมดา ขอบตาดำจากการทำงานหนักและอดนอน แต่เมื่อเอาจริงจะมีแววตาที่เยือกเย็น
    *   *แรงจูงใจ:* หาเงินรักษา "คาริน" น้องสาวที่ป่วยเป็นโรคประหลาด "Data Corruption Syndrome" (ร่างกายค่อยๆ กลายเป็นพิกเซลและหายไป)

### ครอบครัว (Family)
*   **คาริน (Karin):** น้องสาวของคุริโตะ ป่วยหนักด้วยโรคที่รักษาไม่หาย ต้องใช้ "ยาต้านไวรัส" (Antiviral Serum) ราคาแพงลิบฉีดทุกสัปดาห์เพื่อยื้อชีวิต

### ตัวละครสมทบ (Supporting Cast)
*   **คามิ (Kami):** คู่แข่ง/เพื่อน ระดับ Admin สายความเร็ว ชื่อแปลว่า "พระเจ้า" หรือ "จิตวิญญาณ" (的神) เป็นอัจฉริยะที่เย่อหยิ่งแต่ยอมรับในฝีมือ (ในอนาคต)
*   **อาสึมิ (Azumi):** หัวหน้า Server หญิงผู้เข้มงวด เชี่ยวชาญการป้องกัน (Firewall)
*   **ชิน (Shin):** รุ่นน้องผู้มองโลกในแง่ดี (Level 1 User) ชื่อแปลว่า "ความจริง" หรือ "ใหม่"

## 5. จุดเริ่มต้นเรื่อง (Prologue Concept)
โลกเริ่มพังพินาศจาก "The Blue Screen Event" เมื่อ 10 ปีก่อน คุริโตะ ดีบักเกอร์ Level 0 ผู้ตกอับ ต้องดิ้นรนทำงานเสี่ยงตายใน Dead Sector เพื่อหาเงินซื้อยาให้น้องสาว จนกระทั่งวันหนึ่งในภารกิจทำความสะอาด Sector ธรรมดาๆ เขาและปาร์ตี้กลับหลุดเข้าไปใน "Hidden Layer" (เลเยอร์ที่ซ่อนอยู่) ที่โหดหินเกินระดับของพวกเขา... และนั่นคือจุดเริ่มต้นของการ "Format" ชีวิตใหม่

## 6. Canon ล่าสุดจากบท 1-2
*   **สถานที่เปิดเรื่อง:** นีโอ-โตเกียว, เขตฮาจิโอจิและสถานีรถไฟใต้ดินร้างใน Dead Sector D-13
*   **อาการของคาริน:** Data Corruption Syndrome ลุกลามถึงนิ้วมือซ้ายบางส่วน ต้องใช้ **Antiviral Serum / แอนติไวรัล เซรั่ม** ต่อเนื่อง
*   **งานรับจ้างเปิดเรื่อง:** คุริโตะรับงานจากปาร์ตี้ **Red Viper** ในฐานะลูกหาบ เพราะค่าจ้างสูงผิดปกติ 20,000 เครดิต
*   **Red Viper (ชั่วคราว):**
    *   **ไทกะ:** หัวหน้าทีม Level 2 Moderator สายโจมตีระยะประชิด
    *   **เคนจิ:** มือปืนเวกเตอร์
    *   **มิกะ:** สายสอดแนม/อาวุธลวดตัดข้อมูล
    *   **โมริ:** ลูกหาบชราที่ถูกจ้างมาพร้อมคุริโตะ
*   **ความจริงของ D-13:** ภารกิจเป็นกับดักเพื่อ "ล่อ" การปรากฏตัวของบอสใน Hidden Layer ชื่อ **The Nest**
*   **บอสแรก:** **Queen Vampire Glitch** ระดับคุกคามเทียบเท่า Level 3 Admin
*   **ความสามารถพิเศษของคุริโตะ:** มองเห็นเส้นข้อมูล/ตัวเลข 0 และ 1 ของโลกชัดผิดปกติ โดยเฉพาะเมื่อเกิด Anomaly หรือ Hidden Layer
*   **Hook ปลายบท 2:** เกิดข้อความปริศนาสีแดง `Critical Exception: Compatible anomaly found.` และคำสั่ง `Execute recovery patch?`

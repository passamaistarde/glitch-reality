# Glitch Reality (โปรเจกต์: เลกันต์)

**สถานะ:** กำลังเขียน (Ongoing)  
**ผู้แต่ง:** AI Assistant (ร่วมกับคุณ NUT)  
**Theme:** Cyber-Occult / System Error / Dark Fantasy
